📋 Project Overview

  • Wallet-only authentication (SUI Testnet)
  • End-to-end encryption with wallet signatures
  • Decentralized storage via Walrus Protocol
  • On-chain file registry on SUI blockchain
  • Google Drive import feature (Web2 to Web3 bridge)
  • React + Vite frontend with Framer Motion animations
  • Express.js backend with AES-256 encryption
  • Tatum SDK integration for blockchain operations
