## What is WalrusVault?

Decentralized file storage with SUI wallet authentication.

**Key Features:**
- 🔐 Wallet-only authentication (no passwords)
- 🔒 End-to-end encryption with wallet signature
- ⛓️ On-chain file registry (SUI blockchain)
- 🐋 Decentralized storage (Walrus Protocol)

---

## How to Use

1. **Open Website:** https://ramdanse55.github.io/walrusvault/
2. **Connect Wallet:** Click "Connect SUI Wallet" button
3. **Upload Files:** Select files to encrypt and store
4. **Download Files:** Access your encrypted files anytime
5. **Logout:** Clear session and return to login

---

## Tech Stack

- **Frontend:** React + Vite + Framer Motion
- **Backend:** Node.js + Express
- **Blockchain:** SUI Network (Testnet)
- **Storage:** Walrus Protocol
- **Encryption:** AES-256-CBC

---

## Security

- Files encrypted with wallet signature
- Only wallet owner can decrypt
- Cross-wallet isolation enforced
- No passwords stored
