### Tweet 6/10 - SECURITY
**SECURITY MODEL - THREE LAYERS**

🛡️ Layer 1: **AES-256-CBC encryption** (client-side)
🛡️ Layer 2: **Wallet signature verification** (every request)
🛡️ Layer 3: **On-chain ownership proof** (SUI blockchain)

Private keys never leave your wallet. Server never sees plaintext. Zero-knowledge architecture. 🔒

### Tweet 8/10 - API ENDPOINTS
**API ENDPOINTS**

📤 **Upload:**
POST /api/upload → Encrypt → Walrus → Register on-chain

📤 **Share:**
POST /api/sharing/grant → Create AccessGrant on SUI

📊 **Analytics:**
GET /api/tatum/balance → Query via Tatum RPC

All authenticated with **wallet signatures**. 🔑
